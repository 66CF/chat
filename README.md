# RK

基于 MiMo API 的 AI 聊天应用，支持语音合成、角色扮演、小游戏等功能。

## 功能

- AI 聊天对话
- 语音合成 (TTS)
- 语音转文字 (STT)
- 角色扮演模式
- 小游戏（海龟汤、真心话大冒险、故事接龙）
- 换装系统
- 日记系统
- 音乐播放
- 表情包
- 喂食互动
- 长期记忆（Google AI embedding）

## 部署

直接部署到 Vercel 即可，无需构建步骤。

## 配置

首次使用需要配置：

- **MiMo API Key** — 用于聊天和语音功能
- **Google AI API Key**（可选）— 用于长期语义记忆
